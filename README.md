# 🏄 El Paredón Surf Forecast

Personal surf/weather dashboard for Guatemala's famous right-hand point break.

## Tech Stack
- **Frontend**: Vanilla HTML/CSS/JavaScript
- **API**: Open-Meteo Marine Weather API
- **Hosting**: GitHub Pages

## Features
- Real-time wave height, wind speed, temperature
- Colored indicators for wave quality (green/orange/red)
- Horizontally-scrolling hourly forecast
- Offline-ready via browser caching
- PWA installation support

## Deployment
1. Push to a public repository
2. Enable GitHub Pages in Settings → Pages
3. Share the generated URL as a QR code!
